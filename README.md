# Expense Tracker

Full-stack expense tracker built using:

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Authentication

## Features

- User Registration
- User Login
- Protected Routes
- Add Income
- Add Expense
- Dashboard Summary
- Transaction History

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Shadcn UI

Backend:
- Express.js
- Prisma
- PostgreSQL (Neon)

## Setup

Frontend:

npm install
npm run dev

Backend:

npm install
npm run dev
